# MoodByte 🧠💙  
Aplicación móvil para el control y mejora de la salud mental

## 📌 Descripción del proyecto
MoodByte es una aplicación móvil diseñada para apoyar el bienestar emocional de las personas mediante herramientas simples, accesibles y efectivas. La aplicación ayuda a gestionar el estrés, la ansiedad y los hábitos personales, ofreciendo un espacio seguro donde el usuario puede comprender sus emociones, desahogarse y mejorar su salud mental día a día.

El proyecto surge como respuesta al aumento del estrés y la ansiedad en la sociedad actual, donde el acceso a ayuda profesional no siempre es posible o asequible.

---

## 🎯 Objetivos
- Ayudar al usuario a comprender y controlar su estado emocional.
- Fomentar hábitos diarios de autocuidado mental.
- Reducir el estrés y la ansiedad mediante ejercicios guiados.
- Ofrecer un espacio seguro de reflexión personal.
- Facilitar el seguimiento del progreso emocional del usuario.

---

## 🧠 Problema que resuelve
Cada vez más personas lidian con estrés, ansiedad y sobrecarga emocional, especialmente en ámbitos como el académico y laboral. Muchas no saben cómo canalizar correctamente estas emociones y no siempre cuentan con apoyo profesional.

MoodByte actúa como una herramienta de acompañamiento emocional, ayudando a sobrellevar los días más pesados de forma más llevadera y reconfortante.

---

## ⭐ Propuesta de valor
MoodByte es una aplicación diseñada para apoyar la salud mental de manera accesible y cercana. A través de un registro emocional, un diario guiado, ejercicios de relajación, rutinas de autocuidado y contenido especializado, la app acompaña al usuario día a día para reducir el estrés, mejorar hábitos y construir una vida emocionalmente más equilibrada.

A diferencia de otras aplicaciones, MoodByte ofrece una experiencia simple, calmada y completa, adaptada tanto a usuarios jóvenes como adultos.

---

## 🧩 Funcionalidades principales
- Registro diario del estado de ánimo con gráficas semanales y mensuales.
- Diario emocional guiado con preguntas sugeridas.
- Ejercicios de relajación y meditación guiada.
- Rutinas diarias de autocuidado con tareas marcables.
- Música relajante adaptada al estado de ánimo.
- Acceso a artículos y contenidos relacionados con la salud mental.

---

## 👥 Usuario objetivo
Personas que:
- Sufren estrés, ansiedad o agotamiento emocional.
- Desean mejorar sus hábitos y su bienestar mental.
- Buscan una herramienta sencilla, intuitiva y accesible.
- Necesitan un apoyo diario sin sustituir la ayuda profesional.

## 🛠️ Tecnologías utilizadas
- Desarrollo multiplataforma
- Lenguaje / framework: Java, Kotlin, C#
- IDE: IntelliJ, Android Studio, VisualStudio code
- Control de versiones: Git y GitHub
- Base de datos: MySql
- Diseño UI/UX enfocado en simplicidad y calma visual

---

## 📊 Metodología de trabajo
El proyecto MoodByte se ha desarrollado en equipo siguiendo una planificación de 10 semanas, organizada mediante un diagrama de Gantt.  
La fase de desarrollo se realiza de forma paralela entre los miembros del equipo, mientras que la documentación se mantiene activa durante todo el proyecto.

El control de versiones se gestiona mediante GitHub, utilizando ramas para el desarrollo de funcionalidades y pull requests para su integración.
